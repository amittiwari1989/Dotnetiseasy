Textarea

A TEXTAREA tag can create a multiple line text box.It has NAME, ROWS and COLS atrributes. ROWS define the no. of rows and COLS define the no. of colums. It ia a non-empty tag.

Example
<TEXTAREA ROWS=2 COLS=30 NAME="about you">
</TEXTAREA>

We can give by default text in textarea.
<TEXTAREA ROWS=10 COLS=50 NAME="about you">
Your favorite games

Other information if you want to give.


</TEXTAREA>
