Select Tag

This tag is used to creat a menu control. This is a non-empty tag.

Example
CityFavorite color
<SELECT NAME=CITY>
<OPTION>Pune
<OPTION>Mumbai
<OPTION>Delhi
<OPTION>Kolkata
<OPTION>Chenni
<OPTION>Goa
<OPTION>Banglor
</SELECT>		 <SELECT NAME=COLOR SIZE=3 MULTIPLE>
<OPTION>Green
<OPTION>Yellow
<OPTION SELECTED>Red
<OPTION>Blue
<OPTION>Orange
<OPTION>Purple
<OPTION>Brown
</SELECT>
City   Favoeite color  
We can decide the size of option display. MULTIPLE is used to select more then one option and SELECTED is fof by default selection.
