Lists

There are two types of List
1.Ordered List
2.Unorderd List Ordered List
Script
<OL>
< LI>Mango
<LI>Orange
<LI>Grapes
<LI>Banana</OL>		 Outcome

  1. Mango
  2. Orange
  3. Grapes
  4. Banana
To change the numeric list syntax is
Script
<OL TYPE=A>
< LI>Mango
<LI>Orange
<LI>Grapes
<LI>Banana</OL>
Outcome
  1. Mango
  2. Orange
  3. Grapes
  4. Banana
