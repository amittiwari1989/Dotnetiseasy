Bold Underline and Italic

These tags are non-empty tags. The Bold tag <B> make the text appear Bold. The Underline tag <U> Underline the text. The Italic tag <I> gives a slant effact to the text.
Ex:-
<FONT SIZE=+2 COLOR=BLUE FACE="Century Schoolbook"><U><I>Font Size +2, color, Font Face is Century Schoolbook, Underline and Italic</I></U></FONT>

Font Size +2, color, Font Face is Century Schoolbook, Underline and Italic
